# PMP: Brainstorm

**Announce at start:** "Using pmp:brainstorm to explore this idea."

Collaborative design stage. Explores the problem space, proposes approaches, and produces a design document — the input for plan generation.

**Always ask before transitioning.** Use AskQuestion. Never auto-advance.

Use agent teams (Task tool) and track progress with TodoWrite throughout.

## Workflow

1. **REQUIRED:** Read [config.md](../pmp/config.md) for current constants
2. **REQUIRED:** Read [brainstorm.md](references/brainstorm.md) and follow it completely — it contains the full brainstorming loop
3. When the user approves the design, ask: "Ready to generate the implementation plan?"
4. If yes → tell the user to invoke `/pmp:plan`
